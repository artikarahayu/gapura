Telegram Chat ID
Untuk dapat menerima notifikasi via Telegram, silahkan koneksikan telegram anda dengan mengikuti salah satu dari 2 instruksi berikut:
Cara 1 (Tonton Video Tutorial )
  1. Install aplikasi Telegram di HP anda. (Klik disini apabila anda tidak bisa install, login, atau membuat akun telegram)
  2. Buka Telegram, cari bot dengan username @PastiPaskabBot atau klik Link Berikut atau scan QR code berikut dengan aplikasi telegram anda.
    @PASTIPASKABBOT
  3. Klik /start
  4. Ketikkan daftar, lalu anda akan mendapatkan pesan yg berisi Chat ID anda
  5. Masukkan Chat ID yang anda dapatkan pada input dibawah ini.
  6. Klik tombol "Uji Coba" untuk mengirimkan pesan uji coba ke telegram anda.
  7. Jika pesan berhasil terkirim, klik tombol "Hubungkan Telegram" untuk menyimpan Chat ID anda.
Cara 2
  1. Install aplikasi Telegram di HP anda.
  2. Klik tombol "Generate QR Code" disamping.
  3. Scan QR Code yang muncul dengan QR scanner di HP anda yg terinstall telegram.
  4. Anda akan dialihkan ke aplikasi telegram (klik tombol "START BOT" apabila muncul).
  5. Apabila berhasil, anda akan mendapatkan pesan yang menginformasikan tentang akun anda.
  6. Refresh halaman ini. Telegram anda sudah terhubung ke akun gapura emas anda dan selanjutnya anda akan menerima notifikasi aplikasi melalui telegram.
